Bu bir **kontrol odası** tab işidir (bkz. `docs/kontrol-odasi/README.md` — bu tab kod yazmaz; yalnızca `docs/` + Linear + analiz). Bu konuşma bir kod yazma tab'ıysa, "bu kontrol odası işi, doğru tab'ta mıyız?" diye sor; teyit alınca devam et.

Görev: **Linear ↔ docs senkronu** (`doc-denetim`'in dar/hızlı versiyonu).

1. Linear MCP ile Pusula projesindeki tüm issue'ları çek.
2. Karşılaştır: `docs/process/05-is-kayit-defteri.md` (satırlar) ve `docs/process/02-mvp-faz-plani.md` (faz + alt-iş statüleri).
3. Farkları listele: eksik defter satırları, durum kaymaları, sahip/faz uyuşmazlıkları, durum lejantı ile Linear state seti arası fark (`Backlog` / `Sonraki Faz` / `Todo` / `In Progress` / `Done` / `Canceled`).
4. Kullanıcı onayıyla `docs/process/05-is-kayit-defteri.md` + `docs/process/02-mvp-faz-plani.md`'yi hizala (eksik satırları ekle, durumları güncelle). Linear tarafında düzeltme gerekiyorsa (ör. yanlış state) onay alıp uygula.
