# 02 — Teknoloji Kararları

> Eksen: **tasarım / teknik**. Bu dosya **kararların kaydıdır** — yerleşik kabul edilir;
> kullanıcı açıkça istemedikçe yeniden açma. Karar değişirse aşağıdaki "Karar kaydı"na tarihli satır ekle.
