### Misafir sayfası

**Konum**: `apps/web/src/app/share/[token]/`

- **`page.tsx`** (yeni) — SSR App Router:
  - server-side fetch `${env.NEXT_PUBLIC_API_URL}/share/${token}` → 200/404/410
  - 404 → `notFound()` (Next.js standart 404 sayfası)
  - 410 → custom error component (reason'a göre Türkçe metin: revoked/expired/cardArchived/cardDeleted)
  - 200 → snapshot render (kart başlık + açıklama Tiptap content + etiketler + üyeler + checklist + yorumlar + attachment listesi)
  - HTML `<head>`: `<meta name="robots" content="noindex,nofollow">`
  - app-shell DEĞİL — sade public layout (`layout.tsx` ayrı tutulabilir)
- **`_components/share-card-view.tsx`** (yeni) — kart snapshot UI:
  - Üst bilgi: "Bu kart sizinle paylaşıldı" + workspace adı + paylaşan kişi
  - Tiptap read-only render (`useEditor({ editable: false })`) açıklama için — referans: `card-detail-description.tsx` + `rich-text-helpers.ts`
  - Etiketler + üyeler (ad + avatar, e-posta yok) + checklist (read-only)
  - Yorumlar: misafir yorumları `strings.share.guest.guestAuthorLabel` ("Misafir") ile resolve
  - "Realtime yok — yenilemek için sayfayı yenileyin" uyarısı
- **`_components/share-comment-form.tsx`** (yeni) — misafir editor:
  - Şu an için `textarea` + submit; Tiptap mention-disabled editor V2'ye bırakılabilir
  - `POST ${API_URL}/share/${token}/comments` (`fetch` client-side)
  - 10KB body sınırı (client-side validation + server zaten reddediyor)
  - Optimistic ekleme yok; submit sonrası router.refresh()
- **`error.tsx`** (yeni) — Next.js error boundary
- **CSP/header**: SSR response'unda `Referrer-Policy: no-referrer` header set et (`next.config` veya middleware)

## Bilinen takip işleri (Faz 9 sonrası)

1. **`.env.example` + `apps/api/src/env.ts` Zod schema'da `APP_URL` validate** — 9B/9C için W2: production deploy'da `APP_URL` unutulursa link `http://localhost:3000/share/...` olur. Ayrı `chore(env)` commit veya Faz 11 hardening.
2. **Misafir attachment indirme presigned URL endpoint** — 9C'de `storageKey` snapshot'tan kaldırıldı (security review P1). 9D/V2: `GET /share/:token/attachments/:id` endpoint'i kısa süreli S3 presigned URL döndürür.
3. **Rate limit Redis-backed** — şu an in-memory token bucket; multi-instance deployment'ta bucket paylaşılmaz (Faz 8 hardening).
4. **`TRUSTED_PROXY_IPS` env kontrolü** — `x-forwarded-for` spoof koruması (security review P1 dev'de spoofable; production reverse proxy bağımlı — runbook).
5. **Partial index `share_links_card_active_idx WHERE revoked_at IS NULL`** — database-reviewer önerisi; aktif link sorgusu 5-20x daha verimli. Faz 11/12 post-MVP.
6. **Misafir Tiptap rich editor** (mention extension disabled) — şu an textarea ile yeterli; V2'de zenginleştirme.
7. **Faz 9 e2e race condition (Faz 5D paterni)** — share.spec'te socket-join wait kullanılmıyor (misafir realtime yok); ama bob notification bell senaryosunda `data-realtime-board-joined` polling gerekebilir.
